Mini E-Commerce React App

This project is a simple mini e-commerce application built using React to demonstrate
component-based design, state management, and clean coding practices.

Features
- Product listing (15+ products)
- Search products by name
- Filter products by category
- Sort products by price (Low → High, High → Low)
- Add / remove items from cart
- Update item quantity with stock validation
- Cart persistence using localStorage
- Empty state handling (no products, empty cart)

Tech Stack
- React (Functional Components & Hooks)
- JavaScript (ES6)
- CSS (No UI libraries)
